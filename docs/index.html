주요 기능

이런 점이 편합니다

실시간 스트리밍 번역

번역 결과가 문단 단위로 바로 표시되어, 긴 챕터도 앞부분부터 먼저 읽을 수 있습니다.

관심작품

읽고 있는 작품을 등록해 두고, 새 글과 읽은 상태를 한 화면에서 확인할 수 있습니다.

번역 캐시

이미 번역한 문장은 캐시에 남아 있어, 같은 내용을 다시 요청하지 않습니다.

이전/다음 화 이동

번역한 챕터에서 바로 앞뒤 화로 이동해 읽는 흐름을 이어갈 수 있습니다.

다양한 LLM 지원

Gemini, Claude, OpenAI, OpenRouter, 또는 직접 지정한 엔드포인트까지. 제공자·모델·프롬프트·치환 규칙을 앱 안에서 바꿀 수 있습니다.

내보내기

번역 결과를 TXT 또는 HTML로 저장해 두면 인터넷 없이도 읽을 수 있습니다.

FAQ

자주 묻는 질문

API 키가 필요한가요?
네, 번역에 사용할 LLM 제공자의 API 키가 필요합니다. Gemini, OpenRouter, OpenAI 등에서 API 키를 발급받아 설정에 입력하면 됩니다. OpenAI (Codex)를 선택하면 ChatGPT 로그인으로도 사용할 수 있습니다.
어떤 LLM을 추천하나요?
번역 품질과 비용은 모델마다 다릅니다. 처음이라면 짧은 챕터 하나로 모델과 프롬프트가 마음에 드는지 확인한 뒤 긴 챕터를 번역하는 편이 안전합니다.
번역 비용이 얼마나 드나요?
TransNovel 자체는 무료 오픈소스입니다. 비용은 선택한 LLM 제공자의 API 요금에 따라 달라집니다. 같은 문장을 다시 번역하지 않도록 캐시가 동작하므로 비용을 줄이는 데 도움이 됩니다.
번역 결과를 오프라인으로 볼 수 있나요?
네, 번역된 챕터를 TXT 또는 HTML로 내보내면 인터넷 없이도 읽을 수 있습니다. 내보내기는 각 챕터 화면에서 가능합니다.
macOS에서 "확인되지 않은 개발자" 경고가 뜹니다
앱을 처음 실행할 때 macOS Gatekeeper가 경고를 띄울 수 있습니다. 시스템 설정 > 개인 정보 보호 및 보안에서 "확인 없이 열기"를 선택하거나, 터미널에서 xattr -cr /Applications/TransNovel.app을 실행하면 됩니다.
지원하지 않는 사이트를 추가할 수 있나요?
현재 Syosetu, Hameln, Kakuyomu, Nocturne을 지원합니다. 추가 사이트 요청은 GitHub Issues에 남겨 주세요.
